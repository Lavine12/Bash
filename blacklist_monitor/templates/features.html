Application Features

This page provides an overview of what the blacklist monitor can do.

  • Add or remove single IPs or CIDR ranges.
  • Add or remove DNSBL domains (bulk import supported).
  • Create groups and assign IPs using checkboxes. When a group is deleted the IP addresses are not removedâ€”they are placed back in the ungrouped list so you can reassign them later.
  • Manual blacklist check for a single IP.
  • Adjustable check schedule via the web UI.
  • Periodic checks run according to the configured schedule.
  • Alerts are sent to a Telegram chat when an IP is blacklisted.
  • Dashboard shows last and next check times along with blacklist results.
  • Backup past check results with searchable history and configurable schedule. Old backups are cleaned up automatically based on your retention settings for days or total count.
  • View streaming application logs with a configurable history size (default 200 entries).
